<p>Um projeto fullstack feito em React, onde se consumiu uma API pública do digimon. O projeto é um site onde o usuário para ter acesso a o conteúdo da API deve criar uma conta no site e logar em seguida. Depois de logado o usuário vai na aba search e no campo de busca ele pode digitar o nome do digimon e após ele apertar o botão de busca se o digimon que ele digitou o nome existe é exibido em tela para ele um card contendo o nome do digimon,a imagem, o nível e o número dele.</p>
